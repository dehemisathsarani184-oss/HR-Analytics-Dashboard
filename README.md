## HR Analytics Dashboard

This project is an HR Analytics Dashboard created using Power BI. It helps visualize important HR related metrics such as monthly income, age group, gender, attrition, employee satisfaction, job roles, performance ratings, and tenure groups. The dashboard supports HR managers and decision makers by providing clear insights into workforce patterns.

---

## üìå Project Overview

The aim of this project is to analyze employee data and present meaningful insights through interactive data visualizations.  
Using Power BI, the dashboard provides a simple and clear view of HR metrics that help organizations improve decision making.

---

## üéØ Objectives

- To understand employee distribution based on job role, tenure, marital status, gender and age group.  
- To analyze average monthly income, total employees, satisfaction scores and performance ratings.  
- To identify attrition patterns by salary slabs and job roles. 
- To analyze employee education fields and job levels. 
- To create an interactive and user-friendly Power BI dashboard.

üìä Key Findings

- The largest number of employees are in the 25-35 age group.
- The largest number of employees are in the experienced tenure group.

- The sales department generates the highest monthly income.
